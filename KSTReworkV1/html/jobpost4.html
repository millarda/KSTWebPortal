System Administrator


Responsible for designing, organizing, modifying, supporting a company's computer systems. Designs and installs LANs, WANs, Internet and intranet systems, and network segments Support LANs, WANs, network segments, Internet, and intranet systems.

Primary Responibilities


  • Maintain system efficiency and performance.
  • Ensure design of system allows all components to work properly together.
  • Troubleshoot problems reported by users.
  • Make recommendations for future upgrades.
  • Maintain network and system security.
  • Analyze and isolate issues pertaining to systems and applications.
  • Monitor networks to ensure security and availability to specific users
  • Evaluate and modify system's performance.
  • Identify user needs.
  • Maintain integrity of the network, server deployment, and security.
  • Ensure network connectivity throughout enterprise LAN/WAN infrastructure is on par with technical considerations.
  • Deploy and configure networks and components.
  • Assign configuration of authentication and authorization of directory services.
  • Maintain network facilities in individual machines, such as drivers and settings of personal computers as well as printers.
  • Maintain network servers such as file servers, VPN gateways, and intrusion detection systems.
  • Install and configure computer systems, software applications, security updates and patches.
  • System image creation and deployment.
  • Administer servers, desktop computers, printers, routers, switches, firewalls, phones, personal digital assistants, and smartphones.
  • Implement, operate, and maintain network security devices.
  • Work with security engineering team to integrate new capabilities, plan lifecycle replacements, and provide input into the security architecture roadmaps.
  • Maintain, configure and troubleshoot Linux based appliances and servers to include McAfee IDS/IPS.
  • Create programmatic documentation, standard operating procedures and granular process checklists.
  • Manage the functionality and efficiency of a group of computers running on one or more operating systems.
  • Maintain the integrity and security of servers and systems.
  • Sets up administrator and service accounts.
  • Maintain system documentation
  • Interact with users and evaluates vendor products.
  • Make recommendations to purchase hardware and software, coordinates installation and provides backup recovery.
  • Develop and monitor policies and standards for allocation related to the use of computing resources.
  • Program in an administrative language.
  • Develop and implement testing strategies and document results.
  • Provides advice and training to end-users.
  • Provides guidance and work leadership to less-experienced staff members.
  • Maintains current knowledge of relevant technologies as assigned.
  • Participates in special projects as required.

Experience in managing/troubleshooting the following products and services:


  • VMware Virtual Infrastructure- ESX Server 4.1 or newer, Virtual Center Server 4.1 or newer, Virtual Client, Virtual Machines
  • Windows Environment – Windows 2003/2008/2012 Server. Microsoft Exchange Server 2007 or newer. Microsoft SQL Server 2005 or newer. Microsoft Windows 7 Professional. Experience in managing Microsoft Active Directory Domain; Microsoft Exchange and SMTP Services; Windows System Update Server (WSUS) ; Group Policy management; DHCP.

Understanding and experience in:


  • Cisco switches /routers * setup and configuration.
  • HTTP/SMTP gateway appliance and filtering devices such as IronPort.
  • NETAPP Equipment and Technologies- Snap Mirror, CIFS, NFS
  • Virtual Technologies* VMware
  • Must have an active Department of Defense Security Clearance

Certifications


  • CompTIA A+, Network +, and Security+
  • Successful Candidates should have or be able to certify on Microsoft Windows Server 2008 Enterprise Administrator 70-647
