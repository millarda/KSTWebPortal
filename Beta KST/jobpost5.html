System Administrator


The System Administrator will be responsible for, managing, configuring and troubleshooting various VMware, Microsoft and NetApp OS and NOS products including Virtual Center Server, VMware Infrastructure Client, VMotion, Microsoft Server, Microsoft Exchange Server, Microsoft SQL Server, Software Update Services, Microsoft SharePoint, Windows XP, Windows 7, Microsoft Office Suite, NetAPP Data on Tap, Snapdrive for Windows and SnapManager for Exchange.

Responsibilities:


  • Plan and schedule the installation of new or modified hardware, operating systems, and software applications.
  • Administer account, network rights, and access to systems and equipment.
  • Troubleshoot complex system problems as necessary.
  • Administer system resources, the installation and integration of system fixes and workarounds, updates, and enhancements including performance, capacity, availability, serviceability and recoverability.
  • Develop and oversee new policies and standards for server performance monitoring.
  • Administer system hardware, operating systems and software systems to ensure network access through DISN Video Services Global (DVS-G)
  • Administer system hardware, operating systems and software systems to ensure network access through Video Information Exchange Services (VIXS)
  • Must also have extensive, in depth knowledge of various hardware platforms including servers, blade center solutions, storage area networks, Simple Mail Transfer Protocol (SMTP) and Hypertext Transfer Protocol (HTTP) proxies.

Qualifications:


  • Active DoD Top Secret Clearance.
  • CompTIA Security + (CE) Certification.
  • Must be a US Citizen.
  • Must Speak English.
  • 5 Years of Experience.
