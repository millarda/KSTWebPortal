Systems Administrator (DLA)


Ka’ala Systems Technology Corp. (KST) is looking for a Systems Administrator who can work with a team to support the Defense Logistics Agency (DLA) located at Joint Base Pearl Harbor/Hickam Hawaii. Prior Military experience is desirable, but not mandatory. Candidate will join a KST team and provide Windows Server System Administration support for all DLA systems to include software/hardware configuration, security patches implementation, application interfaces, troubleshooting, consultation and new workload planning to the variety of current and new customers.

Skills:



  • Understand system administration to include Server Virtualization
  • Understand system administration to include Server Virtualization
  • Experience with applying DISA STIGs
  • Troubleshoot and resolve IT problems across the enterprise.
  • Tests, locates, and repairs equipment problems associated with trouble calls, and performs routine maintenance.
  • Maintains current knowledge of relevant technologies as assigned.
  • Active Directory Administration. System Administrator provides support for the implementation and configuration of Global Policy Objects (GPO) within the DLA active directory domain.
  • VMware ESXi. System Administrator provides support for the implementation, management and configuration of virtualization technologies including VMware ESXi software within the environment.
  • Storage Area Network Administration. System Administrator provides support for implementation, configuration and management of Storage Area Network with Netbackup
  • The DLA environment: includes Windows 2008 Server, HP UNIX, Windows 7, VMware ESXi Server, Microsoft Internet Information Server, Microsoft System Center Control Manager, Microsoft Exchange, Netbackup, Microsoft Office,

Experience:


  • Two years of experience.
  • Knowledge of DoD IT systems, to include Windows, and systems administration concepts.

Mandatory Minimum Requirements & Certifications:


  • Security+
  • Microsoft Certified Solutions Associate- Server (2008 or higher)
  • U.S. citizenship certification.
  • Active DoD SECRET security clearance.
