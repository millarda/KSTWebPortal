System Administrator


 We are looking for a Systems Administrator who can work with a team to plan the future growth and expansion of the Navy Expeditionary Combat Command (NECC) network and systems at the Expeditionary Service Support Centers (ESSC) located in Construction Battalion Centers Port Hueneme, CA. Prior Military Tactical Telecommunications experience is desirable.

Responsibilities:


 Provide System Administrator support in maintaining, configuring and troubleshooting various VMware, Microsoft, Red Hat Linux (RHEL 5/6), NetApp and Network Operating System (NOS) products included in approved ATO accreditation boundaries for each enclave, including: Virtual Center Server, VMware Infrastructure Client, VMotion, Microsoft Server, Microsoft Exchange Server, Microsoft SQL Server, Software Update Services, Microsoft SharePoint, Windows 7/10, Microsoft Office Suite, NetAPP Data on Tap, Snapdrive for Windows and SnapManager for Exchange.

  • The System Administrator will be a part of a System Support team responsible for maintaining the current systems operational, IA compliant and keeping the IT support staff updated on the latest technologies, tools and processes.
  • Administer & troubleshoot user accounts. Administer network accounts, network rights, and access to systems and equipment. Administer system resources, installation and integration of systems fixes and workarounds, updates and enhancements, including; performance, capacity, availability, serviceability and recoverability.
  • Plan and schedule installation of new or modified hardware, operating systems and software applications.
  • Assist with the develop and review new policies and standards for server performance monitoring and improvement.
  • Participates in special projects as required.
  • Travel may occur 1 - 2 times a year to Hawaii, Japan or Guam

Skills:


  • Host-based Security Software
  • System administration 3 years
  • Understand network/systems design skills to include Server Virtualization, Network Security Design and Microsoft Systems Design
  • VMware products
  • Experience with applying DISA STIGs
  • Troubleshoot and resolve IT problems across the enterprise
  • Knowledge of various hardware platforms, including; servers, blade center solutions, storage area networks, SMTP and HTTP proxies
  • Tests, locates, and repairs equipment problems associated with trouble calls, and performs routine maintenances
  • Maintains current knowledge of relevant technologies as assigned

Preferred Requirements:


  • 2 years’ experience
  • Prior Military Tactical Telecommunications or System Administration experience is desirable
  • CompTIA Security+ CE or the equivalent 8570 certifications

Mandatory Requirements:


  • U.S. citizenship certifications.
  • Active TOP SECRET or SECRET security clearance.
